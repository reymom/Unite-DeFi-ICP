# Relayer Integration Test: Orderbook Pull + Price Tick and Dutch Auction

This test verifies that the relayer canister successfully polls the orderbook, imports auction orders, and computes a price tick.

---

## 1. Buid, Deploy Orderbook Canister

As described in [1.%20orderbook-test.md](1.%20orderbook-test.md).

## 2. Deploy the relayer

```bash
KEY="<alchemy-api-key>"

(
```

```bash

candid-extractor target/wasm32-unknown-unknown/release/relayer.wasm > backend/relayer/relayer.did
dfx deploy relayer --argument-file tmp/relayer_init.did
```

## 3. Deploy an order with the secret

Use a value slightly in the past so auction is already live:

```bash
NOW=$(date +%s)
```

Compute a secret for testing purposes (not keccack256):

```bash
#!/bin/bash
set -euo pipefail

FRIENDLY="my friendly secret phrase"
SECRET_HEX=$(printf '%s' "$FRIENDLY" | openssl dgst -sha3-256 -binary | xxd -p -c 256)
HASHLOCK_HEX=$(printf '%s' "$SECRET_HEX" | xxd -r -p | openssl dgst -sha3-256 -binary | xxd -p -c 256)
secret_vec=$(python3 - <<PY
h = bytes.fromhex("$SECRET_HEX")
print("; ".join(str(b) for b in h))
PY
)

hashlock_vec=$(python3 - <<PY
h = bytes.fromhex("$HASHLOCK_HEX")
print("; ".join(str(b) for b in h))
PY
)
```

```bash
cat > tmp/submit_order.did <<EOF
(

    order_hash = "0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa";

    maker_asset = variant { ICP };
```

## 6. Verify check

This needs interaction with escrows, the resolver needs to deploy ICP escrow via the factory canister and deploy the EVM escrow using chain's fusion evm calls. However we can test the relayer does not accept incorrect escrows.

```bash
ORDER_HASH="0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"
RESOLVER_EVM="0xC0412cD5c995dee4260812c135D93f1dAd53648c"
EVM_ESCROW="0xdeadbeefdeadbeefdeadbeefdeadbeefdeadbeef"
ICP_ESCROW="aaaaa-aa"

dfx canister call relayer verify_and_reveal_secret '(
  "'"$EVM_ESCROW"'",
  "'"$ICP_ESCROW"'",
  "'"$ORDER_HASH"'",
  "'"$RESOLVER_EVM"'"
)'
```

> To see how to deploy the factory canister to deploy the ICP leg escrow, check [4.%20escrow-factory-test.md](4.%20escrow-factory-test.md)
