## Demo – Manual Resolver Flow

Resolver acts as a proxy bot to interact with the relayer, engage automatically in auctions and win rewards in behalf of the canister. It runs completely onchain thanks to ICP's chain fusion, connected to evm and easily extensible to solana and bitcoin.

We can also deploy it without automatic configuration, and use it to manually deploy the escrows on evm and withdraw the funds once we have the secret:

### 0 Build canister

```bash
cargo build -p resolver --release --target wasm32-unknown-unknown
candid-extractor target/wasm32-unknown-unknown/release/resolver.wasm  > backend/resolver/resolver.did
```

### 1 Deploy resolver (without auto-tick)

```bash
dfx identity use taker
cat > tmp/resolver_init.did <<EOF
(
  record {
    relayer_id  = principal "$(dfx canister id relayer)";
    factory_icp = principal "$(dfx canister id factory)";
    evm = record {
      ecdsa_key_id = record { name = "dfx_test_key"; curve = variant { secp256k1 } };
      chains = vec {
        record { 421614; record { rpc_urls = vec { "https://arb-sepolia.g.alchemy.com/v2/$KEY" } } }
      };
    };
    automatic_tick = null;
  }
)
EOF
```

```bash
dfx deploy resolver --argument-file tmp/resolver_init.did
```

> For the automatic relayer, we need to set up the `automatic_tick` parameter to work for us:

```bash
automatic_tick = opt record {
  max_slippage_bps = 100 : nat16;        # 1 %
  poll_interval_sec = 30  : nat64;       # every 30 s
  min_profit_icp    = 1_000_000 : nat64; # 0.01 ICP
  filters = record {
    allow_assets = vec {};               # empty → accept all
    deny_assets  = vec {};
  };
};
```

### 2 Deploy escrows manually

```bash
dfx canister call resolver deploy_evm_escrow '(
  "0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"
)'
```

### 3 Withdraw funds

On the evm leg, the resolver controlled by the taker executes:

```bash
SECRET_HEX="b0950b2960a7070bc50a3ded6fd65abf44058b32938d9302d63827a3b9694731"

dfx canister call resolver withdraw_evm_manual '(
    "0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa",
    blob "0x'$SECRET_HEX'"
)'
```
